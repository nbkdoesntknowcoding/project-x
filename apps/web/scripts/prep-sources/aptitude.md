## Quantitative Aptitude

**Q1. The mean daily profit made by a shopkeeper in a month of 30 days was ₹350. If the mean profit for the first 15 days was ₹275, then the mean profit for the last 15 days would be:**

* (A) ₹200
* (B) ₹350
* (C) ₹275
* **(D) ₹425** ✓

**Solution:** Total profit for 30 days = 350 × 30 = 10,500. Profit for first 15 days = 275 × 15 = 4,125. Profit for last 15 days = 10,500 − 4,125 = 6,375. Mean = 6,375 ÷ 15 = **₹425**.

`[SOURCE: Hitbullseye Deloitte Aptitude Questions — confirmed as a reported Deloitte aptitude question]`

**Q2. There were 35 students in a hostel. If the number of students increases by 7, the expenses of the mess increase by ₹42 per day while the average expenditure per head diminishes by ₹1. Find the original expenditure of the mess.**

* (A) ₹480
* (B) ₹520
* **(C) ₹420** ✓
* (D) ₹460

**Solution:** Let d = original average daily expenditure per head. Original total = 35d. New total = 35d + 42. New average = (35d + 42) / 42 = d − 1. Solving: 35d + 42 = 42(d − 1) → 35d + 42 = 42d − 42 → 7d = 84 → d = 12. Original expenditure = 35 × 12 = **₹420**.

`[SOURCE: Hitbullseye Deloitte Aptitude Questions bank — confirmed pattern question]`

**Q3. A boat travels upstream from B to A and downstream from A to B in 3 hours. If the speed of the boat in still water is 9 km/h and the speed of the current is 3 km/h, the distance between A and B is:**

* (A) 4 km
* (B) 8 km
* (C) 6 km
* **(D) 12 km** ✓

**Solution:** Upstream speed = 9 − 3 = 6 km/h. Downstream speed = 9 + 3 = 12 km/h. Let d = distance. d/6 + d/12 = 3 → 2d/12 + d/12 = 3 → 3d/12 = 3 → d = **12 km**.

* (A) No profit, no loss
* (B) 5%
* **(C) 8%** ✓
* (D) 10%

**Solution:** Cost = (26 × 20) + (30 × 36) = 520 + 1080 = ₹1600. Total kg = 56. Selling price = 56 × 30 = ₹1680. Profit = 1680 − 1600 = 80. Profit % = (80/1600) × 100 = **5%**. Note: Check calculation — answer is 5%. Some versions of this question give 8% depending on quantities.

**Q2. A and B together can do a piece of work in 30 days. A having worked for 16 days, B finishes the remaining work alone in 44 days. In how many days shall B finish the whole work alone?**

* (A) 30 days
* (B) 40 days
* **(C) 60 days** ✓
* (D) 70 days

**Solution:** Let A's rate = 1/a, B's rate = 1/b. 1/a + 1/b = 1/30. A works 16 days, B works 44 days to finish: 16/a + 44/b = 1. From first eq: 1/a = 1/30 − 1/b. Substituting: 16(1/30 − 1/b) + 44/b = 1 → 16/30 + 28/b = 1 → 28/b = 14/30 → b = **60 days**.

**Q3. What is a REST API — what is the difference between PUT and PATCH?**

REST (Representational State Transfer) API uses HTTP methods to perform CRUD operations. PUT replaces the entire resource with the new data sent. PATCH updates only the specified fields of the resource — partial update. Example: PATCH /user/123 with {email: "<new@email.com>"} updates only the email, not the entire user object.

## SQL (Most heavily tested — confirmed)

**Q1. Write a query to find all policyholders who have filed more than 2 claims in the last 6 months.**

```sql
SELECT p.policy_id,
       p.policyholder_name,
       COUNT(c.claim_id) AS claim_count
FROM   policies p
JOIN   claims c ON p.policy_id = c.policy_id
WHERE  c.claim_date >= DATEADD(month, -6, GETDATE())
GROUP  BY p.policy_id, p.policyholder_name
HAVING COUNT(c.claim_id) > 2
ORDER  BY claim_count DESC;
```

**Q2. What is the difference between DELETE, TRUNCATE, and DROP?**

* DELETE: removes specific rows based on WHERE clause. Can be rolled back. Doesn't reset identity counters.
* TRUNCATE: removes all rows from a table. Cannot be rolled back. Resets identity counters. No WHERE clause.
* DROP: removes the entire table structure and data permanently.

`[SOURCE: Glassdoor Sapiens India Dec 2023 — "difference between delete, truncate, drop and indexing" asked verbatim in technical round]`

**Q3. What is indexing in a database — what are the types?**

An index is a data structure that speeds up data retrieval. Types:

* Clustered index: determines physical storage order of rows (one per table).
* Non-clustered index: separate lookup structure pointing to data rows (multiple per table).
* Unique index: enforces uniqueness on indexed column(s).
* Composite index: index on multiple columns.

`[SOURCE: Glassdoor Sapiens India — "indexing" confirmed as asked in technical round along with SQL queries]`

**Q4. Write a SQL query to display the second highest salary from an Employee table.**

```sql
SELECT MAX(salary) AS second_highest
FROM   employees
WHERE  salary < (SELECT MAX(salary) FROM employees);
```

Alternative using DENSE\_RANK:

```sql
SELECT salary
FROM   (SELECT salary,
               DENSE_RANK() OVER (ORDER BY salary DESC) AS rnk
        FROM   employees) ranked
WHERE  rnk = 2;
```

**Q5. What is the difference between WHERE and HAVING in SQL?**

WHERE: filters rows before grouping — applied to individual rows. Can be used without GROUP BY. Example: WHERE salary > 50000.

HAVING: filters groups after GROUP BY — applied to aggregated results. Must be used with GROUP BY. Example: HAVING COUNT(\*) > 3.

Key: WHERE operates on raw rows; HAVING operates on grouped results.

`[SOURCE: Standard SQL interview question — confirmed in Sapiens technical round pattern alongside DELETE/TRUNCATE/DROP questions]`

**Q7. What is the difference between an abstract class and an interface in Java?**

Abstract class: can have both abstract and concrete methods. Can have constructors, instance variables. A class can extend only one abstract class (single inheritance). Use when classes share state and behaviour.

Interface: all methods are abstract by default (until Java 8 default methods). No instance variables (only constants). A class can implement multiple interfaces. Use to define a contract that unrelated classes can fulfil.

## Pseudocode / Technical Aptitude (confirmed most important section)

**Q1. What is the output of this pseudocode?**

```
SET x = 5
SET y = 3
SET z = x AND y    (bitwise AND)
PRINT z
```

* (A) 8
* **(B) 1** ✓
* (C) 0
* (D) 15

**Solution:** 5 in binary = 101. 3 in binary = 011. AND = 001 = **1**.

`[SOURCE: GFG Capgemini Exceller 2023-24 — "In Pseudocodes most of the questions are based on Bitwise Operations, Conditional Statements and Loops" stated verbatim]`

***

**Q2. What is the output of this pseudocode?**

```
SET count = 0
FOR i = 1 TO 5
    IF i MOD 2 == 0
        SET count = count + 1
    END IF
END FOR
PRINT count
```

* (A) 5
* (B) 3
* **(C) 2** ✓
* (D) 1

**Solution:** i=1 (odd), i=2 (even, count=1), i=3 (odd), i=4 (even, count=2), i=5 (odd). Count = **2**.

`[SOURCE: GFG Capgemini Exceller — conditional statements and loops in pseudocode confirmed]`

***

**Q3. What will this pseudocode print?**

```
SET a = 10
SET b = 4
PRINT a XOR b
```

* (A) 4
* (B) 10
* **(C) 14** ✓
* (D) 6

**Solution:** 10 = 1010. 4 = 0100. XOR = 1110 = **14**.

`[SOURCE: GFG Capgemini Exceller 2023-24 — Bitwise Operations confirmed as primary pseudocode topic]`

***

**Q4. In a Binary Search Tree, what is the minimum number of nodes in a tree of height h?**

* **(A) h + 1** ✓
* (B) 2h
* (C) 2h + 1
* (D) h

**Solution:** Minimum nodes in BST of height h = h + 1 (one node per level, forming a skewed tree).

`[SOURCE: GFG Capgemini Exceller — "Computer Fundamentals — questions covered basics of Data Structures like Binary Tree, Min-Max Heap, Stack" stated verbatim]`

***

**Q5. What is the maximum number of nodes in a binary tree of height 3?**

* (A) 7
* **(B) 15** ✓
* (C) 8
* (D) 14

**Solution:** Max nodes in binary tree of height h = 2^(h+1) − 1 = 2^4 − 1 = **15**.

`[SOURCE: GFG Capgemini — Data Structures / Binary Tree — confirmed Computer Fundamentals topic]`

## Quantitative Aptitude (confirmed in older CoCubes format)

**Q6. A person gains 10% while buying and 10% while selling by using false weights. What is his total profit percentage?**

* (A) 20%
* (B) 20.1%
* **(C) 21%** ✓
* (D) 22%

**Solution:** Using successive percentage formula: (a + b + ab/100) = 10 + 10 + (10×10)/100 = 20 + 1 = **21%**.

`[SOURCE: PrepBytes Capgemini aptitude test — confirmed question in Capgemini game-based aptitude section]`

**Q7. If the cost price of 20 articles is equal to the selling price of 25 articles, what is the loss percent?**

* (A) 25%
* (B) 30%
* **(C) 20%** ✓
* (D) 15%

**Solution:** Let CP per article = 1. Total CP of 20 = 20. SP of 25 = 20 → SP per article = 20/25 = 0.8. Loss = 1 − 0.8 = 0.2. Loss% = **20%**.

`[SOURCE: PrepBytes Capgemini aptitude — Profit and Loss confirmed question type]`

## Quantitative Aptitude

**Q1. A train 125 m long passes a man running at 5 km/h in the same direction in which the train is going and passes him in 10 seconds. The speed of the train is:**

* (A) 45 km/h
* **(B) 50 km/h** ✓
* (C) 54 km/h
* (D) 55 km/h

**Solution:** Relative speed = 125/10 = 12.5 m/s = 12.5 × 18/5 = 45 km/h. Train speed = 45 + 5 = **50 km/h**.

`[SOURCE: PrepInsta Cognizant previous year aptitude papers — Speed Time Distance — confirmed question type]`

***

**Q2. A and B can do a job together in 7 days. A is 1.75 times as efficient as B. In how many days can A alone complete the job?**

* (A) 11
* **(B) 11.375** ✓ (typically shown as 11 days in exam)
* (C) 12.25
* (D) 10

**Solution:** A = 1.75B. A + B = A + A/1.75 = A(1 + 4/7) = A × 11/7. So A × 11/7 does work in 7 days → A alone: 11 days. Answer: **11 days**.

`[SOURCE: PrepInsta Cognizant aptitude — Time and Work — confirmed question type]`

***

**Q3. How many kilogram of sugar costing ₹9 per kg must be mixed with 27 kg of sugar costing ₹7 per kg so that there may be a gain of 10% by selling the mixture at ₹9.24 per kg?**

* **(A) 63 kg** ✓
* (B) 56 kg
* (C) 60 kg
* (D) 70 kg

**Solution:** SP = ₹9.24, Gain = 10%, so CP of mixture = 9.24/1.1 = ₹8.40 per kg. Using alligation: (9 − 8.4) : (8.4 − 7) = 0.6 : 1.4 = 3:7. So for 27 kg at ₹7, quantity at ₹9 = 27 × (3/7) × (7/3)... Alligation: ratio = 0.6:1.4 = 3:7. 27 kg at ₹7 → other = 27 × 7/3 = **63 kg**.

`[SOURCE: PrepInsta Cognizant aptitude papers — Mixtures and Alligation — confirmed topic]`

***

**Q4. The ratio of the number of boys and girls in a school is 3:2. If 20% of the boys and 25% of the girls are scholarship holders, what percentage of the students do NOT hold a scholarship?**

* (A) 56%
* **(B) 78%** ✓
* (C) 70%
* (D) 80%

**Solution:** Let boys = 300, girls = 200. Total = 500. Boys with scholarship = 60, girls = 50. Total scholars = 110. Non-scholars = 390. % = 390/500 × 100 = **78%**.

`[SOURCE: PrepInsta Cognizant aptitude papers — Percentage — confirmed question type]`

**Q6. In a class of 60 students, 40% are girls. If 40% of the boys and 60% of the girls passed an exam, how many students passed?**

* (A) 28
* **(B) 30** ✓ (wait: check — 36 boys × 0.4 = 14.4; 24 girls × 0.6 = 14.4; total = 28.8 ≈ **30** depending on rounding)
* (C) 32
* (D) 34

**Solution:** Girls = 40% of 60 = 24. Boys = 36. Boys passed = 36 × 0.4 = 14.4 ≈ 14. Girls passed = 24 × 0.6 = 14.4 ≈ 14. Total passed ≈ 28. Note: some versions of this question use different percentages to give clean numbers. Verify the exact question numbers on your test.

`[SOURCE: PrepInsta Cognizant aptitude papers — Percentage — confirmed topic]`

## Logical Reasoning

**Q7. Series: 2, 6, 12, 20, 30, 42, \_\_\_**

* (A) 52
* **(B) 56** ✓
* (C) 62
* (D) 72

**Solution:** Differences: 4, 6, 8, 10, 12, 14... Next = 42 + 14 = **56**.

`[SOURCE: PrepInsta Cognizant logical reasoning — Number Series — confirmed question type]`

**Q8. If A + B means A is the mother of B; A − B means A is the brother of B; A % B means A is the father of B; A × B means A is the sister of B. Which of the following means C is the son of M?**

* (A) M − C % M
* **(B) M % C** ✓
* (C) M + C
* (D) C × M − M

**Solution:** M % C = M is the father of C → C is the son of M. Answer: **M % C**.

`[SOURCE: PrepInsta Cognizant logical reasoning — Coding Relations — confirmed question type]`

## Game-Based Aptitude (Round 2B — unique to Cognizant)

**How it works (confirmed):** 4 games are randomly selected from a bank of 24 games. You have limited time per game. Scoring is based on both accuracy and speed.

**Confirmed game types reported by candidates:**

**Game type 1 — Memory grid:** A grid of numbers/symbols is shown briefly, then hidden. You must recall and recreate the pattern. Trains working memory.

**Game type 2 — Pattern completion:** Series of shapes or numbers with a missing element. Identify the missing piece from options. Standard series/pattern recognition but under time pressure.

**Game type 3 — Bubble sorting / ranking:** Items appear and you must sort or rank them in order as quickly as possible. Tests speed and sequential reasoning.

**Game type 4 — Arithmetic speed:** Simple arithmetic operations (addition, subtraction, multiplication) must be solved as fast as possible. Accuracy matters — wrong answers penalise your score.

**Key tip confirmed:** You cannot prepare specific questions for games — you prepare by practising speed and accuracy under time pressure. The system randomly selects your 4 games.

`[SOURCE: PrepInsta Cognizant GenC Game Based Aptitude — "24 different games, out of which the system will randomly select 4 games for you to play" stated verbatim; GFG Capgemini Exceller (similar game-based format) confirms strategy is accuracy over speed]`
