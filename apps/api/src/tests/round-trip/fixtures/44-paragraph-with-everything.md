# Paragraph with everything inline

A single paragraph that mixes **bold**, *italic*, ~~strikethrough~~, `inline code`, an [inline link](https://example.com), and inline math $E = mc^2$ all together. The point is that any one of these inline marks could conflict with another's parsing if the schema is loose; the round-trip must preserve the structure exactly.

A second paragraph that adds an autolink <https://example.org> and an image ![alt](https://example.com/x.png) into the same dense pattern.
